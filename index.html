angular新建组件方法

方法一

工具：vs code

项目名：project-app

我的组件名为header

步骤：

  • vs code打开project-app项目

  • 打开src文件夹

  • 进入app文件夹

  • 在app文件夹中新建一个文件夹作为组件文件夹

  • 在文件夹中创建两个文件命名为 name.component.ts(name为你的组件名称)、name.component.html
