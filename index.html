angular新建组件方法

方法一

工具：vs code

项目名：project-app

步骤：

  • vs code打开project-app项目

  • 打开src文件夹

  • 进入app文件夹

  • 在app文件夹中新建一个文件夹作为组件文件夹

  • 在文件夹中创建两个文件命名为 name.component.ts(name为你的组件名称)、name.component.html

    示例组件如下

  • 在name.component.ts文件中数据以下内容

    import { Component } from “@angular/core”;

    import { Template } from ‘@angular/compiler/src/render3/r3_ast’;

    @Component({

    ​ selector:’app-name’,(name为你的组件名称)

    ​ templateUrl:’./name.component.html’此处为你的组建中新建的html文件的路径

    })

    export class HeaderComponent{

    ​

    }

    示例代码

    import { Component } from “@angular/core”;

    import { Template } from ‘@angular/compiler/src/render3/r3_ast’;

    @Component({

    ​ selector:’app-header’,

    ​ templateUrl:’./header.component.html’

    })

    export class HeaderComponent{

    ​

    }

  • 找到app.module.ts文件，将nameCompontent添加到@NgModule中，同时在文件头部添加import { NameComponent } from ‘./header/header.compoment’; 此处是你组件的路径

    示例代码

    import { BrowserModule } from ‘@angular/platform-browser’;

    import { NgModule } from ‘@angular/core’;

    import { AppComponent } from ‘./app.component’;

    import { HeaderComponent } from ‘./header/header.compoment’;

    @NgModule({

    declarations: [

    ​ AppComponent,

    ​ HeaderComponent,

    ],

    imports: [

    ​ BrowserModule

    ],

    providers: [],

    bootstrap: [AppComponent]

    })

    export class AppModule { }

组件新建成功，可以开始在组件中添加内容了

方法二：命令行方式

  • 在cmd中cd到project-app文件夹
  • 输入ng g c header即可创建组件，自动完成了以上步骤
